1. Entry

1.1. What is my program?

MC Code Copier Reloaded is a Python program that can copy reward codes in Anarchia.GG Minecraft Server (OneBlock) and let you receive your reward every time. Every 30 minutes, server is sending message with 10-characters long code (e.g. LrQ9ihPNlq). My program's role is to capture and parse it.

1.2. How does it work?

This program have to scan in-game chat to look for new codes. Hopefully, Minecraft is logging all chat messages to latest.log. Program is instantly scanning it and looking for new codes. When it detects this scheme:

Plain text
Przepisz kod <code> aby otrzymać nagrodę!

it instantly knows that new code appeared. Now, it can do many things specified in config file: Copy it to clipboard, send notification, and more.

When it detects this message, it starts scanning and looking for reward info scheme. When it detects this scheme:

Plain text
Gracz <player name> jako pierwszy przepisał kod w czasie <re-write time> i otrzymał(a) 3 Klucze AFK!

It is logging reward info and it can save it to many formats.

Installation

2.1. Python

Program is written in Python, so it needs Python to run. In some systems, Python is installed by default, but in others, you need to install it manually.

NOTE: This documentation only shows steps for Windows and Linux. Unfortunately (or fortunately - I hate Apple), I don't have access to any Apple PC to write here about installing Python.

First, you need to check is Python already installed on your system. To do it, open terminal (on Windows - search for PowerShell in start menu; on Linux - search for Terminal) and type following command:

PowerShell / Bash
python3 --version

If this command returns something like this (version may differ):

Plain text
Python 3.14.5

then you can go to here and continue configuring everything. If it returned some red text (on Windows) or Command not Found (on Linux), then we need to install it.

2.1.1. Windows

To install Python on Windows, type following commands:

PowerShell
winget update
winget install python3

And finish the installation. If command finished, restart it and run:

PowerShell
python3 --version

Now it should print your exact Python version.

2.1.2. Linux

To install Python on Linux, you need to use package manager for specific distro:

Bash
# Ubuntu/Debian:
sudo apt update
sudo apt install python3
# Arch Linux:
sudo pacman -Sy
sudo pacman -S python3
# Fedora/Red Hat Enterprise Linux:
sudo dnf update
sudo dnf install python3

Then, you can run this command:

Bash
python3 --version

To confirm installation success.
